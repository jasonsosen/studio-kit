# Studio Kit

Desktop content management tool with AI-powered text generation.

## Features

- Calendar view for scheduling
- AI text generation (Claude API)
- Weekly planning
- Auto-launch on startup

## Development

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Tech Stack

- React + TypeScript + Tailwind CSS
- Rust + Tauri 2.0
- SQLite
- Claude API
